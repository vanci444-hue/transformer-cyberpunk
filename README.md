# Transformer · Cyberpunk Edition

A single-page visual walkthrough of the Transformer architecture, styled as a cyberpunk diagram.

## Overview

`transformer-cyberpunk` is a self-contained HTML page that explains how a Transformer works — tokens, embeddings, attention, feed-forward, stacked blocks — through a guided visual narrative rather than dense equations.

It is built to be the kind of thing a product manager or designer can open and actually come away understanding, without a prerequisite ML course.

## Why it exists

Most Transformer explainers fall into two camps: overly technical (papers and blog posts aimed at researchers) or overly hand-wavy (marketing diagrams). I wanted a third option: visually serious, conceptually correct, and approachable for non-researchers.

## Features

- Step-by-step visualization of the Transformer forward pass
- Cyberpunk-inflected design — high contrast, neon accents, intentionally opinionated
- Single HTML file, no dependencies to install
- Works as a course supplement or a standalone explainer
